Connect your WooCommerce store with Make Influence quickly.

== Description ==

Install this plugin and connect your WooCommerce store with the Make Influence platform. This plugin is compatible with WooCommerce 3.1+ and works with all major payment providers.

== Installation ==

1. Upload the plugin's folder to the `/wp-content/plugins/` directory or install it through the plugin installer

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Navigate to Make Influence

4. Insert your Business ID
